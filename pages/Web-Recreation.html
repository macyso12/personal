BU Internal Student Web-Recreation - Team Leader

This project is inspired because of the old-fashioned style internal student link interface. As a transfer student, I have to relearn everything when I get to a larger scale university like Boston University. After using the Fordham's internal student link website, and changing into the BU student link website, I got confused when I first clicked onto the site. My first impression is that it is not user-friendly, adn the UXUI design has a old-fashion style and does not create a good impression for me. And that inspired me to do a personal project on how I can recreate BU Student Link Website to make it a more user-friendly and intuitive website than it is now.
